# React Tag Input Component

This project is a simple React application built with Vite that demonstrates a Tag Input component.

Users can:
- Add tags by typing in the input field and pressing Enter
- Remove tags by clicking the close button

## Tech Stack

- React
- Vite
- JavaScript
- CSS

## Project Setup

Clone the repository:

```bash
git clone <repo-url>
```

Navigate to project:

```bash
cd tag-input-react
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```
